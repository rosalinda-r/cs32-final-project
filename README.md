# cs32-final-project
Final Project Goal: Hangman Game With Graphics
For our final project we would like to create computer vs. player game of Hangman. We will be using server and client logic, where the client is always guessing.
We also want to expand our abilities of generating visuals by having a visual element to our game.
Our subtasks would include:
    1. Having the server pick a random word
    3. Taking a person's guess for each of the letters in the word, in sequence
    4. Changing our visual based on weather or not the person got the letter right or wrong
    5. Looping the game till the player wins or loses
